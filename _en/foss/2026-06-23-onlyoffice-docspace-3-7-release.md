## The Open Source Collaboration Platform Evolves

ONLYOFFICE DocSpace is an open source document collaboration platform, licensed under the GNU Affero General Public License v3.0 [citation:1][citation:14]. It allows teams to collaboratively edit documents, spreadsheets, presentations, forms, and PDF files online.

On June 8, 2026, ONLYOFFICE announced DocSpace 3.7, a major update that touches nearly every part of the platform [citation:1]. This release includes new AI-powered features, a reworked form filling experience, updated document editors, improved file and room management, enhanced security settings, and a unified Billing section [citation:1][citation:6].

🔗 **Official Source:** [onlyoffice.com/blog/2026/06/onlyoffice-docspace-3-7](https://www.onlyoffice.com/blog/2026/06/onlyoffice-docspace-3-7)
🔗 **Official Website:** [onlyoffice.com](https://www.onlyoffice.com)

## Generate Files Directly from AI Chats

One of the most exciting features in version 3.7 is the ability to **generate DOCX, PDF, and PPTX files directly from AI agent chats** [citation:1][citation:9]. Users can now, while chatting with an AI agent, request the creation of a text document, presentation, or PDF form. The agent generates the file and automatically opens it in a new tab for editing [citation:1].

For self-hosted users, activating this feature requires enabling the Automation API, which is currently available upon request through the sales team [citation:1].

## AI Agents: More Flexibility and New Providers

DocSpace 3.7 introduces support for new AI providers, expanding user options:

- **New providers**: DeepSeek, Google AI, and xAI, in addition to existing providers (Anthropic, OpenAI, OpenRouter, Together AI, and custom providers) [citation:1][citation:9].
- **Default provider synchronization**: The default provider selected in DocSpace settings can now be automatically synchronized to the editors, eliminating the need for separate AI configuration in each place [citation:1].
- **Image uploading in chat**: Users can now upload images from DocSpace directly into the AI chat, providing visual context for tasks [citation:1].
- **Extended thinking mode**: The chat can display an extended thinking section in some responses, depending on request complexity [citation:1].
- **Disable AI when needed**: DocSpace administrators can now completely disable AI features across DocSpace and the editors, while preserving chat history [citation:1].

## Reworked Form Filling Experience

Form Filling rooms received a major update in version 3.7, with new ways to create, manage, and collect forms [citation:1]:

- **Connect to external databases**: Responses can now be collected in an external database in addition to the built-in spreadsheet. Connect your database via Settings → Integration → Third-party services [citation:1].
- **Create and edit forms inside the room**: Users are no longer required to upload a ready-made form. They can now create a new PDF form from scratch or from a text file directly inside the room [citation:1].
- **New form filling mode**: Forms must now be switched to filling mode using Start filling before they become available for room members to complete. This prevents unintentional editing [citation:1].
- **Update collected data**: A new "Sync responses to XLSX" option allows refreshing the Excel file where form responses are gathered [citation:1].

## Editors Powered by Docs 9.4

Version 3.7 includes the latest version of ONLYOFFICE editors (Docs 9.4) with improvements [citation:1]:

- Horizontal lines for better document structure.
- Dark Document mode in spreadsheets.
- 25 new ready-to-use slide themes.
- 20 new slide transitions.
- A dedicated Chart Design tab.
- Croatian interface support.

## Improved File and Room Management

- **Replace default template files**: DocSpace administrators can now replace the files used as templates for creating new documents, via Settings → Customization → Default templates [citation:1].
- **Group rooms**: Rooms can be combined into groups, with group tags displayed below the search field. Multiple rooms can also be deleted at once [citation:1].
- **Manage room tags**: Add, delete, and change room tags quickly.

## Security Enhancements

- **Prohibit external links**: Administrators can prevent the creation of external links to protect sensitive content (Settings → Security → DocSpace access) [citation:1].
- **Invite link controls**: Set limits on how many users can join via an invite link and how long the link remains valid (Contacts → Invite via link → Link settings) [citation:1].

## Unified Billing

The payments and services pages are now unified under a single "Billing" section, bringing together the tariff plan, Wallet, payment method, and services such as extra disk storage and backup [citation:1][citation:6].

## Other Quality-of-Life Improvements

- **Search filtering**: Separate search by owner.
- **Owner change details**: Displayed in the history info panel.
- **Choose startup section**: Select which section DocSpace opens to.
- **Context menu hotkeys**: Open the context menu using keyboard shortcuts.
- **Redesigned document template library**.
- **Stop ongoing backups**.
- **Blocked users hidden from selector**.
- **Login to DocSpace via Nextcloud**.
- **Fixed header on smartphones** when scrolling up [citation:1].

## Availability and Download

DocSpace 3.7 is directly available in the cloud. New users can create a free account, and existing users can sign in to try the new features. For self-hosted versions, the update will be available soon [citation:1].

## Summary

ONLYOFFICE DocSpace 3.7 is a comprehensive update that strengthens the platform's position as an integrated, open source collaboration solution. The AI integration for file generation, new flexibility in form management, and security improvements make this release a powerful choice for teams seeking a practical alternative to Microsoft Office or Google Workspace, while maintaining full control over their data.
