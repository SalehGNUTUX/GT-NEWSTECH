## What is GT-IPNET?

GT-IPNET is a GNUTUX project that provides a unified graphical interface for common Linux networking tools like `ping`, `traceroute`, `nmap`, `whois`, `dig`, and more — no terminal required.

## Key Features

- **Modern UI** built with React + TypeScript + Tailwind CSS v4
- **Secure IPC** communication with the OS via Electron
- **Auto-detection** of installed system packages
- **RTL/LTR support** for bilingual display
- **Dark and light themes** fully integrated

## Tech Stack

| Technology | Version |

| Electron | 28+ |

## Installation

```bash
git clone https://github.com/SalehGNUTUX/GT-IPNET.git
cd GT-IPNET
npm install
npm run dev
```
