## ما هو GT-IPNET؟

GT-IPNET هو مشروع من مشاريع GNUTUX، يهدف إلى توفير واجهة رسومية موحدة لأدوات شبكة لينكس الشائعة مثل `ping`، `traceroute`، `nmap`، `whois`، `dig`، وغيرها، دون الحاجة إلى سطر الأوامر.

## المميزات الرئيسية

- **واجهة حديثة** مبنية بـ React + TypeScript + Tailwind CSS v4
- **تواصل IPC** مع النظام عبر Electron لتنفيذ الأوامر بأمان
- **اكتشاف تلقائي للحزم** المثبتة على النظام
- **دعم RTL/LTR** للعرض الثنائي اللغة
- **وضع داكن وفاتح** متكاملان

## التقنيات المستخدمة

| التقنية | الإصدار |
|---------|---------|
| Electron | 28+ |
| React | 18+ |
| TypeScript | 5+ |
| Tailwind CSS | 4+ |

## تثبيت المشروع

```bash
git clone https://github.com/SalehGNUTUX/GT-IPNET.git
cd GT-IPNET
npm install
npm run dev
```
