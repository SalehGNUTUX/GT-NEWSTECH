## LibreOffice 24.8 — أبرز المستجدات

أصدرت **The Document Foundation** الإصدار الجديد من مجموعة LibreOffice المكتبية المفتوحة المصدر، والذي يجلب تحسينات ملحوظة في الأداء والتوافق.

## أبرز التحسينات

### تحسينات Writer
- سرعة تحميل الملفات الكبيرة أسرع بنسبة 40%
- دعم محسّن لـ `.docx` مع الحفاظ على التنسيق
- أداة إملاء صوتي جديدة

### تحسينات Calc
- معالجة أسرع للجداول الكبيرة
- دوال إحصائية جديدة متوافقة مع Excel
- تحسين عرض المخططات البيانية

### تحسينات Impress
- قوالب حديثة جديدة
- دعم انتقالات الشرائح المتقدمة

## التوافق مع النظم

LibreOffice متاح على:
- **لينكس**: عبر مستودعات التوزيعة أو Flatpak
- **ويندوز**: MSI أو winget
- **macOS**: DMG

## التحميل

```bash
# Ubuntu/Debian
sudo apt install libreoffice

# Flatpak
flatpak install flathub org.libreoffice.LibreOffice
```
