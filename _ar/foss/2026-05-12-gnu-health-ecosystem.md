## GNU Health: عندما يصبح الطب الاجتماعي برمجيات حرة

**GNU Health** ليس مجرد نظام معلومات صحية – إنه **نظام بيئي صحي رقمي متكامل**، يجمع بين رسالة **الطب الاجتماعي** (Social Medicine) وأحدث تقنيات المعلوماتية الصحية، مع الالتزام التام بفلسفة البرمجيات الحرة.

> "الطب بدون إنسانية طبية لا يستحق أن يمارس." – الأستاذ الدكتور رينيه فافالورو (جَرّاح قلبي أرجنتيني)

🔗 **الموقع الرسمي:** [gnuhealth.org](https://www.gnuhealth.org)

---

## ما هو GNU Health؟

GNU Health هو مشروع من **مشاريع GNU** الرسمية، يهدف إلى تحسين الرعاية الصحية في المجتمعات المحرومة والمناطق النامية، عبر توفير نظام معلومات صحي **متكامل، معياري، وآمن**، يمكن استخدامه في:

- المستشفيات والعيادات
- مراكز الرعاية الأولية
- وزارات الصحة (للإحصاءات والتقارير)
- مختبرات الأبحاث الطبية
- وحدات الرعاية المتنقلة

---

## البنى التقنية تحت الغطاء

| الطبقة | التقنية / المنصة |
|--------|------------------|
| **المحرك الأساسي** | Tryton (نظام ERP مفتوح المصدر، بديل قوي لـ SAP) |
| **اللغة الخلفية** | Python 3 |
| **اللغة الأمامية** | GTK / Web (عبر WebDAV) |
| **قاعدة البيانات** | PostgreSQL (مع دعم PostGIS للمعطيات الجغرافية) |
| **ترخيص الصورة** | GPLv3+ (لكود النظام) و GFDL (للتوثيق) |

> الميزة الأساسية: **GNU Health ليس مجرد تطبيق قائم بذاته – إنه منصة صحية كاملة**، تتضمن نظام معلومات المستشفى (HIS)، ونظام المختبرات، ونظام الصيدلة، وسجلات المرضى الإلكترونية (EHR)، ووحدة التطعيمات، والإحصائيات الوبائية.

---

## المكونات الأساسية للنظام البيئي

### 1. GNU Health HMIS (نظام معلومات إدارة الصحة)
- **سجل المريض الإلكتروني (EHR)** – شامل مع التاريخ الطبي والحساسية والأدوية.
- **إدارة المواعيد** – عيادات خارجية وداخلية مع تذكيرات.
- **إدارة الأقسام** (الطوارئ، العمليات، العناية المركزة، إلخ).
- **الفواتير والتأمين الصحي** – دعم لأنظمة الدفع المختلفة.

### 2. GNU Health Laboratory (مختبر GNU الصحي)
- إدارة عينات المرضى وتسليم النتائج.
- تكامل مع أجهزة المختبر الآلية (عبر بروتوكولات HL7 و ASTM).
- نظام مراقبة الجودة الداخلية.

### 3. GNU Health Pharmacy (صيدلية GNU)
- إدارة المخزون الدوائي (تتبع الأدوية من الشراء إلى الصرف).
- تنبيهات حول تواريخ الانتهاء والتفاعلات الدوائية.
- دعم الوصفات الإلكترونية.

### 4. GNU Health Socioeconomics (العوامل الاجتماعية الاقتصادية)
- وحدة فريدة في نظم المعلومات الصحية، تركز على **محددات الصحة الاجتماعية**:
  - السكن، التعليم، الدخل، الوضع الوظيفي.
  - التعرض للعنف المنزلي.
  - الوصول إلى المياه النظيفة والصرف الصحي.
- تفي باحتياجات تقارير **الصحة العالمية** (WHO) حول عدم المساواة الصحية.

### 5. GNU Health Genetics (علم الوراثة)
- دعم السجلات الوراثية للمرضى (أمراض نادرة).
- تكامل مع قواعد بيانات الجينات (Gene Ontology، OMIM).
- خرائط توارث (نسب) للأمراض العائلية.

### 6. GNU Health Federation (الاتحاد الصحي)
- بروتوكول لامركزي لتبادل البيانات الصحية بين المؤسسات المختلفة.
- يضمن الخصوصية (لا يتم تبادل البيانات دون موافقة المريض).
- مبني على تقنيات: **Thalamus** (جهاز توجيه صحي)، و **REST API** موثقة.

## لماذا يعد GNU Health مشروعاً نموذجياً لمجتمع FOSS؟

### 1. يطبق الطب الاجتماعي – فعلاً
ليس مجرد شعار. الوحدة الاجتماعية الاقتصادية **تترجم الفلسفة إلى كود**: تحليل بيانات عدم المساواة الصحية وتأثيرها على المرضى (بدونها تصبح المعلومات الصحية ناقصة).

### 2. يعمل في ظروف قاسية
صُمم للعمل في:
- مناطق إنترنت غير مستقر (يعمل وضع عدم الاتصال بشكل كامل).
- أجهزة قديمة منخفضة الموارد (يدعم حتى أجهزة Pentium III مع 512MB RAM).
- شبكات بطيئة (يتم ضغط البيانات عبر Federation).

### 3. آمن وصارم مع الخصوصية
- التشفير افتراضي لجميع بيانات المرضى (AES-256).
- سجلات تدقيق (audit trails) لكل وصول إلى السجل الطبي.
- الامتثال للائحة العامة لحماية البيانات (GDPR) وقانون نقل المعلومات الصحية (HIPAA) من تصميمه الأساسي.

### 4. مدعوم من مشروع GNU ومجتمع قوي
- جزء من نظام GNU الرسمي – يخضع لمراجعات الجودة والأمان من FSF.
- مجتمع نشط من الأطباء والممرضين والمبرمجين ومترجمي التقارير الطبية.

## التحديات التي يعالجها GNU Health

| التحدي | الحل عبر النظام |
|--------|------------------|

## كيف يمكن البدء باستخدام GNU Health؟

### للمستخدمين الطبيين (مستشفى/عيادة)

#### الطريقة 3: الحزم الثنائية (لينكس)
```bash
# Ubuntu / Debian
sudo apt install gnuhealth

# حزمة التثبيت الشاملة (من مصادر النظام)
# تتبع دليل التثبيت السريع على الموقع الرسمي
```

#### الطريقة 4: الحاوية (Docker) – للإنتاج
```bash
git clone https://codeberg.org/gnuhealth/gnuhealth-docker.git
cd gnuhealth-docker
docker-compose up -d
```

### للمطورين الراغبين في المساهمة

```bash
# استنساخ المستودع الرئيسي
hg clone https://hg.savannah.gnu.org/hgweb/health/
cd health

# تثبيت المتطلبات
pip3 install -r requirements.txt

# تشغيل نظام Tryton
trytond -c trytond.conf
```

> **ملاحظة:** GNU Health يستخدم **Mercurial (hg)** كنظام تحكم بالإصدارات (وليس git). المستودع مستضاف على **Savannah (FSF)**.

## خلاصة

**GNU Health ليس مجرد برنامج – إنه تجسيد رقمي لرؤية "الصحة حق أساسي للجميع".**

يجمع بين:
- **أحدث البرمجيات الحرة** (Python, Tryton, PostgreSQL).
- **أعرق المؤسسات الطبية** (AIIMS، الصليب الأحمر، وزارات الصحة).
- **أهم فكرة في الصحة العامة** (الطب الاجتماعي – محدِّدات الصحة).

إذا كنت مسؤولاً في مؤسسة صحية، أو مطوراً يبحث عن مشروع مفتوح المصدر له تأثير إنساني مباشر، أو طالب طب يريد أن يرى كيف يُطبق "الطب الاجتماعي" في كود حقيقي – فـ GNU Health هو بوابتك.
