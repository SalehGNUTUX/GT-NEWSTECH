## لينكس ومنصة الألعاب — قصة نجاح حقيقية

شهدت السنوات الأخيرة تحولاً جذرياً في عالم الألعاب على لينكس. ما كان يُعدّ حلماً بعيد المنال بات اليوم واقعاً ملموساً، إذ تجاوز عدد الألعاب المدعومة على Steam وحده **20,000 لعبة** تعمل على لينكس بشكل رسمي أو عبر طبقة التوافق **Proton**.

---

## أولاً: Steam و Proton — الثنائي الذي غيّر كل شيء

**Proton** هو مشروع Valve المبني على Wine وDXVK وغيرها من التقنيات، يُتيح تشغيل ألعاب Windows على لينكس بشكل شبه شفاف.

### تفعيل Steam Play (Proton)

```bash
# تثبيت Steam
sudo apt install steam        # Ubuntu/Debian
sudo pacman -S steam          # Arch Linux
flatpak install flathub com.valvesoftware.Steam
```

بعد التثبيت:
1. افتح Steam → **الإعدادات** → **Steam Play**
2. فعّل **"تمكين Steam Play لجميع الألعاب"**
3. اختر أحدث إصدار من Proton

### ProtonDB — معرفة توافق أي لعبة

قبل شراء أي لعبة تحقق من تقييمها على [ProtonDB.com](https://www.protondb.com):

| التقييم | المعنى |
|---|---|
| **Platinum** | تعمل بشكل مثالي |
| **Gold** | تعمل بضبط بسيط |
| **Silver** | تعمل مع بعض المشاكل |
| **Bronze** | تعمل لكن بمشاكل ملحوظة |
| **Borked** | لا تعمل |

## ثانياً: Lutris — مدير الألعاب الشامل

**Lutris** هو منصة مفتوحة المصدر تجمع كل مصادر الألعاب في مكان واحد.

```bash
# تثبيت Lutris
sudo apt install lutris        # Ubuntu/Debian
flatpak install flathub net.lutris.Lutris
```

يدعم Lutris:
- **Steam** — تكامل مباشر
- **Epic Games Store** — عبر Heroic أو مباشرة
- **GOG** — مجتمع يوفر scripts جاهزة
- **Battle.net** — Blizzard games
- **ألعاب Windows** — أي ملف .exe عبر Wine

---

## ثالثاً: Heroic Games Launcher

بديل ممتاز لـ Epic Games Store وGOG على لينكس:

```bash
flatpak install flathub com.heroicgameslauncher.hgl
```

## رابعاً: بطاقات الرسومات وتهيئة النظام

### NVIDIA
```bash
# Ubuntu — تثبيت أحدث Driver
sudo ubuntu-drivers install
```

### AMD (مدمج في kernel)
بطاقات AMD تعمل بشكل ممتاز على لينكس بدون أي تثبيت إضافي — drivers مدمجة في kernel.

### تحسين الأداء — GameMode

```bash
sudo apt install gamemode

# تشغيل لعبة مع GameMode
gamemoderun ./game
```

لينكس في 2026 ليس مجرد خيار ثانٍ للألعاب — إنه منصة ناضجة وقادرة. مع Steam وProton وLutris وHeroic، يمكنك الاستمتاع بآلاف الألعاب بأداء ممتاز. وإذا كنت من مستخدمي بطاقات AMD فأنت محظوظ بشكل خاص.
