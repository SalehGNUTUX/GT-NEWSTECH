## لينكس ومنصة الألعاب — قصة نجاح حقيقية

شهدت السنوات الأخيرة تحولاً جذرياً في عالم الألعاب على لينكس. ما كان يُعدّ حلماً بعيد المنال بات اليوم واقعاً ملموساً، إذ تجاوز عدد الألعاب المدعومة على Steam وحده **20,000 لعبة** تعمل على لينكس بشكل رسمي أو عبر طبقة التوافق **Proton**. مع تحديثات 2026، أصبحت التجربة أكثر سلاسة من أي وقت مضى.

---

## أولاً: Steam و Proton — الثنائي الذي غيّر كل شيء

**Proton** هو مشروع Valve المبني على Wine وDXVK وغيرها من التقنيات، يُتيح تشغيل ألعاب Windows على لينكس بشكل شبه شفاف.

### أحدث إصدار: Proton 10.0-4

في يناير 2026، أصدرت Valve الإصدار المستقر Proton 10.0-4 الذي يضم تحسينات كبيرة:

- **إصلاحات HDR** في ألعاب مثل Far Cry 5 على شاشات OLED
- **تحسين تشغيل ألعاب Xbox Game Studios** بما فيها Avowed وIndiana Jones and the Great Circle
- **دعم أوسع للألعاب** مثل Metal Slug: Awakening وAssassin's Creed Shadows
- **ترقية المكونات الأساسية** تشمل Wine Mono 10.4.1 وvkd3d-proton 3.0b
- **تحسين الـ haptic feedback** لوحدات تحكم DualSense

### تفعيل Steam Play (Proton)

```bash
# تثبيت Steam
sudo apt install steam        # Ubuntu/Debian
sudo pacman -S steam          # Arch Linux
flatpak install flathub com.valvesoftware.Steam
```

بعد التثبيت:
1. افتح Steam → **الإعدادات** → **Steam Play**
2. فعّل **"تمكين Steam Play لجميع الألعاب"**
3. اختر **Proton 10.0-4** كأحدث إصدار مستقر

### ProtonDB — معرفة توافق أي لعبة

قبل شراء أي لعبة تحقق من تقييمها على [ProtonDB.com](https://www.protondb.com):

| التقييم | المعنى |
|---|---|
| **Platinum** | تعمل بشكل مثالي |
| **Gold** | تعمل بضبط بسيط |
| **Silver** | تعمل مع بعض المشاكل |
| **Bronze** | تعمل لكن بمشاكل ملحوظة |
| **Borked** | لا تعمل |

## ثانياً: Lutris — مدير الألعاب الشامل

**Lutris** هو منصة مفتوحة المصدر تجمع كل مصادر الألعاب في مكان واحد. في فبراير 2026، صدر الإصدار **Lutris 0.5.21** بميزات جديدة:

- **دعم محاكيات PS4 وXbox 360** لإدارة ألعاب هذه المنصات
- **دمج Valve Steam Runtime 3.0** لتحسين التوافق
- **محاكي GameCube الآن يعمل كـ AppImage** — لا مشاكل تبعيات بعد الآن
- **تحسين التعريف ببطاقات Intel Arc** تعرض الآن الاسم الصحيح
- **طي الأقسام في الشريط الجانبي** لتنظيم أفضل

```bash
# تثبيت Lutris
sudo apt install lutris        # Ubuntu/Debian
flatpak install flathub net.lutris.Lutris
```

يدعم Lutris:
- **Steam** — تكامل مباشر
- **Epic Games Store** — عبر Heroic أو مباشرة
- **GOG** — مجتمع يوفر scripts جاهزة
- **Battle.net** — ألعاب Blizzard
- **ألعاب Windows** — أي ملف .exe عبر Wine

## ثالثاً: Heroic Games Launcher — إصدار "Pythagoras"

بديل ممتاز لـ Epic Games Store وGOG على لينكس. في فبراير 2026، صدر الإصدار **2.20.0 "Pythagoras"** مع تحسينات:

- **تشغيل ألعاب Epic دون اتصال** للألعاب التي تدعم هذه الميزة
- **إصلاحات للواجهة في اللغات التي تتبع RTL** (مثل العربية)
- **تحسين التعامل مع أخطاء GOG**
- **إصلاح مشكلة النقر المتعدد على زر التشغيل**

```bash
flatpak install flathub com.heroicgameslauncher.hgl
```

## رابعاً: Steam Deck و SteamOS 3.8 — لينكس في راحة يدك

**Steam Deck** من Valve يعمل على **SteamOS** المبني على Arch Linux. في أبريل 2026، أصدرت Valve **SteamOS 3.8.3** (الاسم الكودي "Second Clutch") بتحديثات جذرية:

### دعم موسع لأجهزة الطرف الثالث
- **ASUS ROG Ally/Xbox Ally**: دعم كامل لوحدات التحكم، التحكم بـ TDP، وصوت السماعات
- **Lenovo Legion Go 2**: دعم تحديث البرامج الثابتة، وأضواء RGB، وتحديد حد الشحن
- **تحسين الدعم لأجهزة GPD Win 5، OneXPlayer، MSI Claw وغيرها**

### تحسينات الأداء
- **تأخير إدخال وحدات التحكم** انخفض من 5-8 مللي ثانية إلى **100-500 ميكروثانية** فقط
- **تحسين VRR** و **إصلاح FSR** لا يظهر كخانة في التراكب
- **النواة محدثة إلى Linux 6.16**

### سطح المكتب (Desktop Mode)
- **KDE Plasma محدث إلى 6.4.3** مع **Wayland** كافتراضي
- **دعم شاشات HDR الخارجية** و **شاشات VRR**
- **دعم مقياس منفصل لكل شاشة**

## سادساً: بطاقات الرسومات وتهيئة النظام

### NVIDIA
```bash
# Ubuntu — تثبيت أحدث Driver
sudo ubuntu-drivers install
```

### AMD (مدمج في kernel)
بطاقات AMD تعمل بشكل ممتاز على لينكس بدون أي تثبيت إضافي — drivers مدمجة في kernel. مع SteamOS 3.8، تحسنت إدارة الذاكرة بشكل كبير للمنصات ذات البطاقات المنفصلة (dGPU).

### تحسين الأداء — GameMode

```bash
sudo apt install gamemode

# تشغيل لعبة مع GameMode
gamemoderun ./game
```

## خلاصة

لينكس في 2026 ليس مجرد خيار ثانٍ للألعاب — إنه منصة ناضجة وقادرة. مع Proton 10.0-4، SteamOS 3.8، Lutris 0.5.21، وHeroic 2.20.0، يمكنك الاستمتاع بآلاف الألعاب بأداء ممتاز. وإذا كنت من مستخدمي Steam Deck أو الأجهزة المحمولة الأخرى فأنت محظوظ بشكل خاص مع التحسينات الهائلة في SteamOS 3.8.
