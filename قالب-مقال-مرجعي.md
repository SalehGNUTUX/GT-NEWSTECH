## ما هو Wokwi؟

**Wokwi** مشروع مفتوح المصدر يُتيح محاكاة دارات إلكترونية كاملة
في متصفح الويب — بدون أي تثبيت محلي.

> اقتباس مهم: "أداة ثورية تغيّر طريقة تعلّم الإلكترونيات."  
> — مجلة Embedded Systems

## المميزات الرئيسية

- **محاكاة دقيقة** للأجهزة الحقيقية
- **مكتبة واسعة** من المكوّنات (LEDs، Buttons، Sensors)
- **دعم Arduino** و **MicroPython** و **Rust**
- **مشاركة المشاريع** بنقرة واحدة
- **متعدد الوسائط** بشكل أصلي

### المتحكمات المدعومة

| المتحكم | الدعم | الذاكرة |
|---------|------|---------|
| ESP32   | ✅   | 4 MB    |
| ESP32-S3| ✅   | 8 MB    |
| Raspberry Pi Pico | ✅ | 2 MB |
| Arduino Uno | ✅ | 32 KB |

## كيف يعمل؟

التطبيق يستخدم **WebAssembly** لتشغيل المحاكاة بسرعة قريبة من
الأصلية. الكود يُجمَّع للمنصة الهدف ثم يعمل افتراضياً:

```c
#include <Arduino.h>

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
  Serial.begin(115200);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);
  delay(1000);
  digitalWrite(LED_BUILTIN, LOW);
  delay(1000);
  Serial.println("Blink!");
}
```

## What is Wokwi?

**Wokwi** is an open-source project enabling full circuit simulation
in a web browser — with no local installation required.

> Important quote: "A revolutionary tool changing how we teach electronics."  
> — Embedded Systems Magazine

## Key Features

- **Accurate simulation** of real hardware
- **Wide component library** (LEDs, Buttons, Sensors)
- **Arduino**, **MicroPython**, and **Rust** support
- **One-click sharing**
- **Multi-format** natively

### Supported MCUs

| MCU      | Support | Memory  |
|----------|---------|---------|
| ESP32    | ✅      | 4 MB    |
| ESP32-S3 | ✅      | 8 MB    |
| Raspberry Pi Pico | ✅ | 2 MB |
| Arduino Uno | ✅   | 32 KB   |

## How it Works

The app uses **WebAssembly** to run simulations at near-native speed.
Code is compiled for the target platform then executes virtually:

```c
#include <Arduino.h>

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
  Serial.begin(115200);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);
  delay(1000);
  digitalWrite(LED_BUILTIN, LOW);
  delay(1000);
  Serial.println("Blink!");
}
```

No installation needed — just open the browser:

```bash

cd wokwi-cli

```
